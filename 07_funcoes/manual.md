- o que é uma função
é uma parte do código que pode ser chamada e repetida no meio do codigo quando quiser.
- como declarar uma função
utilizando o comando 
function nomedafuncao() {

}
- como chamar uma função
basta escrever o nome da função. (   Ex: nomedafuncao();  )
- função com parâmetro
permite passar dados para dentro de um bloco de código.
- função com retorno
utiliza a palavra "return" para enviar um valor de volta ao local onde foi chamada, depois de passar pelo código
